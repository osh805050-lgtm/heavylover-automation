## ⚠️ Boost 필요 항목 + 구체적 보강 방향

### #2 참여기업·참여인원의 적합성 (5점, 현재 70%)

**현 상태**: 55~60줄에 4개 불릿만 있음. 표·증빙 부재.

**보강 방향**:
1. **공동대표 R&R 표 추가** — 오승현(전략·재무·광고·CRM 인프라·데이터 분석) vs 박재영(SS·SNS·디자인·CS·물류 운영) 명확히 분리. ACT-01 "박재영 역할 명세 작성(05-06 기한)" 진행 중임을 명시하되, **현 시점 분담은 명확히 정의 가능**.
2. **소상공인 자격 충족 근거 명문화** — "소상공인기본법 제2조: 상시근로자 5인 미만 + 매출 10억 미만 충족" 인용.
3. **참여 인원 (사업 수행 인력) 표** — 본 사업 책임자 / 부책임자 / 지원 인력 명시 (양식상 요구 가능).

### #4 창의성·독창성·사업 성공전략 (15점, 현재 73%)

**현 상태**: "라이프스타일 4축" 잘 다뤘으나 **moat(진입장벽) 약함**. 평가위원 의심 — "다이어트 도시락 회사가 800kcal 라인업 추가하면 끝 아닌가?"

**보강 방향**:
1. **카피 방어 논리 1단락 추가** — (a) 훈제 닭다리살 독립 소싱 = 닭다리살 공급망 자체가 진입장벽 (AI 위기 시 검증), (b) 5개월 누적 ROAS 3.77로 학습된 광고 데이터 자산 = 후발 진입자가 동일 ROAS 도달까지 6~12개월 소요, (c) 카페24+SS 양채널 자동화 = 코드 자산.
2. **"왜 식스밀·가치올림이 800kcal 라인업을 안 만드나" 질문에 답변** — 다이어트 브랜드의 칼로리 일관성 약속(브랜드 톤) 자기 잠식 우려 + 1인분 510g은 다이어트 SKU 대비 콜드체인 비용 ↑.
3. **데이터 인프라 코드 자산 정량화** — Cron 6종 + Python 스크립트 N개 + 자동화 시간 절감 월 N시간.

### #7 사업참여 기대효과·성과창출 가능성 (15점, 현재 73%)

**현 상태**: 정량표 OK. 정성 3개는 추상적. **고용 2→4인 약속이 가장 약함**.

**보강 방향**:
1. **고용 채용 계획 표** — 마케팅 1인 (시점 2026-09, 월급여 ?), CS·물류 1인 (시점 2026-12, 월급여 ?). 채용 트리거 = 월매출 5,000만 도달 or 시리얼 출시 후 60일.
2. **인건비 자기부담 가능성 근거** — 4월 순이익 500만원 → 6,500만원 매출 시 마진 32% × 6,500 = 2,080만 - 고정비 800만 - 광고 600만 = 680만 잉여. 4인 인건비 흡수 가능 추정.
3. **K-Food 수출 카테고리 확장** → "퍼포먼스 푸드" 표현은 자평. **KOTRA 내수기업 수출기업화 합격 사실로 대체** (이미 174줄 인용, 251줄에서는 자평으로 약화됨).
4. **사례화·홍보 가능성** → 강한소상공인 사례화는 평가위원 판단. **본 섹션에서는 "본 사업 종료 후 디지털 전환 사례 콘텐츠 제공 의향"으로 톤다운**.
