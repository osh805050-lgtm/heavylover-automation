## ❌ 제거 또는 즉시 수정 권고 수치 (1차 출처 미확인)

1. **HMR CAGR 8.1%** (TAM) — 농경연 발 1차 출처 미확인. 다른 자료는 17%대 성장 언급. 출처 1차 문서·페이지 명시하거나 "aT FIS / 식품저널 보도(2023-12) 인용"으로 출처 교체 후 CAGR 표현 자체를 보수적으로 수정 권고.
2. **냉동 도시락 4,110억원** (SAM) — 농경연·aT·KOSIS 직접 확인 불가. 4,110억원의 1차 보고서 페이지 번호·캡처 없으면 "도시락 시장 약 2조원 + 냉동 비중 약 X%" 형태로 추정식 분해 후 자체 추정으로 표기 변경 권고.
3. **헬스장 등록자 약 650만** — 문체부 공식 통계 직접 확인 불가 + 본문 v2-draft 124행 "650만"과 CLAUDE.md §4 KPI 본문 §사업·서식의 "600만" 불일치 가능성. 1차 출처 URL 확정 또는 "전국 피트니스센터 약 9,881개(KB금융, 2020)" 등 검증 가능 통계로 교체 권고.
4. **글로벌 F&B ROAS 벤치 1.85~2.0** — 본문 출처 표기 없음. ads.md/benchmarks.md 자체 벤치(평균 2.5)로 교체하거나 외부 벤치 출처(WordStream 등) URL 명시 필수.
5. **더미식 도시락 420kcal·22~26g** — CJ더마켓에 "더미식 도시락" 라인업 SKU명·페이지 직접 확인 불가. 햇반컵반 등 다른 CJ 라인 혼동 우려 → SKU명 명시 또는 페이지 URL 보강 필수.
