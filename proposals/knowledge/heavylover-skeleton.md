## P (Problem) — 문제정의·시장분석 (배점 25)

### 시장 페인 (정량 증거)

운동을 주 3회 이상 하는 20~30대 직장인 남성은 **한 끼당 800kcal 이상 + 단백질 40g**을 챙겨야 벌크업·유지가 가능하다. 그러나 시간(점심 30분, 저녁 회식)·지식(영양 성분 계산) 부족으로 실제로는 단백질 결핍 식사를 반복한다.

- 국내 헬스장 등록자 약 600만 명 (한국체육시설업협회, 2024)
- 운동 식단 관리 의향자 중 실제 실천율 **30% 미만** (헤비로버 자체 설문 N=187, 2025)
- 시판 단백질 도시락 평균 단백질 25g·600kcal → 벌크업 부족

### 창업자 경험 연결고리

대표 오승현(1998년생)은 본인이 헬스 7년차이며, 주 3회 운동 직장인으로서 매일 같은 페인을 겪었다. 시중 닭가슴살은 단조로워 지속 불가능했고, 도시락은 칼로리·단백질이 부족했다. **본인이 가장 큰 고객**.

### 헤비로버가 풀어야 할 문제 (한 줄)

"운동하는 남자가 매끼 800kcal·40g 단백질을 1분 만에 데워 먹을 수 있게 한다."

## S (Solution) — 솔루션·차별화·기술근거 (배점 30)

### 핵심 솔루션

**고칼로리·고단백 냉동 완성식** (1인분 800kcal / 단백질 40g). IQF 급속냉동 + 수비드 저온조리. 전자레인지 1분 30초.

### 4대 차별화 (경쟁사 대비)

| 항목 | 헤비로버 | 잇메이트·닭가슴살닷컴 등 경쟁사 |
|---|---|---|
| 1인분 영양 | 800kcal / 단백질 40g | 평균 600kcal / 단백질 25g |
| 조리 공법 | IQF + 수비드 저온조리 | 일반 냉동·삶음 |
| 원료 | 훈제 닭다리살 (독립 소싱, 단조로움 해결) | 닭가슴살 단일 |
| 인증 | HACCP | 일부만 보유 |

### 기술적 근거

- **HACCP 인증** (식품 안전관리 인증)
- **OEM 파트너 나비야(Navia)** — 검증된 식품 제조 (계약서 첨부 가능)
- **자체 CRM 인프라** — 카페24 + 네이버 스마트스토어 양 채널 OAuth·프록시 자동화 (D2C 식품 동급 희소)
- **실측 데이터 기반 의사결정** — Meta Ads API + 재구매 분석 자동화 가동 중

### 지속가능 우위 (재논의 금지 사안 박제)

- D2C 본질 유지 (B2B 단순 전환 안 함)
- Cafe24 자사몰 + 네이버 스마트스토어 (Imweb 회귀 안 함)
- Lookalike 광고 X, Broad 타겟팅 확정

### 자금 사용 계획 (단가×수량×시점 강제 — 사업별 조정)

자금은 **마케팅·인프라·운영** 3개 축으로만 분류. 각 항목당 단가·수량·시점을 명시. 뭉뚱그림 금지.

예시 (초기창업패키지 1억 가정):
- Meta 광고 집행 6,000만 원 = 월 1,000만 × 6개월 (현재 ROAS 3.77 → 매출 환산 2.26억)
- 시리얼 초도 생산 1,500만 원 = 단가 1,250원 × 12,000개 (OEM 견적 기준)
- CRM·자동화 인프라 1,000만 원 = Vultr·OneDrive·구글워크스페이스 12개월 + 추가 개발 200시간
- 패키지·디자인 800만 원 = 시리얼 봉투 8종 디자인 × 100만 + 시안 비용
- 인건비 700만 원 = 박재영 공동대표 운영 보조 6개월

### 6대 장기 자산 (사업별로 1~3개 골라 강조)

1. **CRM 데이터 인프라** — 동급 D2C 희소. 의사결정 속도 2~3배.
2. **검증된 단위경제학** — ROAS 3.77 + 재구매 30% + AOV 2배 = 광고 스케일 수학적 성립.
3. **PMF 시그널** — 재구매 간격 P50 15일 = 생활 루틴 편입.
4. **명확한 포지셔닝** — 벌크업 냉동 세그먼트 + HACCP.
5. **운영자형 대표** — AI 도구·인프라 직접 구축, 외부 의존 낮음.
6. **양 채널 자동화** — Cafe24 OAuth + SS 프록시 = 희귀 구조.
