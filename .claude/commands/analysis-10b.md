# /analysis-10b

헤비로버 10억 달성 7명 전문가 × 10 Round 심층 분석. 각 Round 결과를 파일로 저장하고 자동 순차 진행.

## 사전 점검
1. `data/analysis_10b/unit_economics.json` 존재 확인
2. `data/meta_ads/daily_campaign.csv` 존재 확인
3. `docs/analysis_10b/rounds/` 디렉토리 확인 (없으면 생성)
4. 오늘 날짜 설정: TODAY=YYYY-MM-DD

사용자 안내: "헤비로버 10억 분석을 시작합니다. 7명 전문가 × 10 Round. 중간 확인 없이 자동 진행. 결과는 docs/analysis_10b/rounds/에 저장됩니다."

## Round 1: 독립 분석
**Devil's Advocate: CMO** (이번 Round에서 합의 의견에 무조건 반대)

강제 질문: "광고를 완전히 끄면 월 매출 얼마 남나? (오가닉+재구매만)"

각 에이전트 순차 호출 (CFO→CMO→COO→고객인사이트→경쟁분석→성장해커→비판):
- 각자 독립 분석 (다른 의견 모르는 상태)
- 형식: 문제 정의 / 핵심 데이터+출처 / 시나리오+수치 / 권고 3개(액션+효과+비용) / 확신도
- CMO는 합의 형성 시 Devil's Advocate 역할 수행
- 비판 에이전트는 6명 발언 후 가정 목록화 + 기각/보류 판정

Round 끝: **핵심 발견 3줄 요약** (간결하게 — 다음 라운드는 이 3줄만 참조) → `docs/analysis_10b/rounds/round-01-independent.md` 저장

## Round 2: 상호 검토 + 반박
**Devil's Advocate: COO**

강제 질문: "7명 중 가장 근거 약한 주장은? 데이터로만 반박하라."

Round 1 요약 참조 후 6명 상호 검토:
- 동의: 이유 명시
- 반대: 데이터/논리로만 반박
- 수정 필요: 구체적으로 어디를 어떻게
- COO: 합의 형성 시 반대 포지션 강제
- 비판: R1 합의 항목 재검증

Round 끝: **핵심 합의 + 핵심 반박 3줄 요약** → `round-02-crosscheck.md` 저장

## Round 3: 비판 에이전트 1차
**Devil's Advocate: 비판에이전트 + CFO**

강제 질문: "10억 시나리오의 치명적 가정 3개는? 각각 검증 방법과 비용은?"

비판 에이전트 주도:
- R1~R2 합의 항목 전체 검토
- 숨은 가정 목록화 (검증 방법 + 비용 포함)
- 실패 시나리오 제시
- CFO는 재무 측면에서 추가 반박

Round 끝: **살아남은 합의 + 보류 합의 3줄 요약** → `round-03-critic-1.md` 저장

## Round 4: 외부 반례 주입
**Devil's Advocate: 고객인사이트**

강제 질문: "한국 D2C 식품 10억 못 간 사례 3개, 10억 이후 무너진 사례 2개. 헤비로버에 적용 시 위험 시그널은?"

경쟁분석 에이전트가 WebSearch로 사례 검색 후 분석:
- 실패 사례 3개 (출처 포함)
- 무너진 사례 2개 (출처 포함)
- 공통 패턴 추출
- 각 에이전트가 자기 분야 패턴 해석
- 고객인사이트: 합의 형성 시 반대 포지션

Round 끝: **헤비로버 위험 시그널 3줄 요약** → `round-04-external-cases.md` 저장

## Round 5: 스트레스 테스트
**Devil's Advocate: 성장해커**

강제 질문: "Meta 효율 -30% 시 6개월 매출 시뮬레이션. 숫자로."

5개 시나리오 각 에이전트 대응 (매출 영향 수치 + 대응 액션 + 회복 기간):
1. Meta 광고비 효율 -30% (ROAS 3.51 → 2.46)
2. 시리얼 6월 출시 지연 or 프리오더 500봉 미달
3. 경쟁사 가격 -20% 인하
4. 정부 지원금 1억 무산
5. 박재영 이탈 (운영 병목)

성장해커: 각 시나리오에서 "그래도 가능한 대안"으로 반박

Round 끝: **가장 위험한 시나리오 TOP 3 + 대응 3줄** → `round-05-stress-test.md` 저장

## Round 6: 가정 역전 토론
**Devil's Advocate: 경쟁분석**

강제 질문: "10억 불가능하다고 가정 시 이유? 현재 추세 360M에서 멈춘다면 원인?"

모든 에이전트 반대 입장 채택:
- "10억 달성 불가능한 구조적 이유"
- "헤비로버만의 고유 약점 (다른 브랜드에 없는 것)"
- 발견된 숨은 가정을 검증 가능한 형태로 변환

경쟁분석: 경쟁사 성공 요인을 헤비로버의 부재 요인으로 재해석

Round 끝: **발견된 숨은 가정 3줄 요약** → `round-06-assumption-flip.md` 저장

## Round 7: 시간 압박 테스트
**Devil's Advocate: CFO**

강제 질문: "1인 운영 + 8개월(2026.5~12) 기준 실행 불가 항목을 솔직히 제거하라."

제약 조건:
- 인력: 오승현 1인 + 박재영 (역할 미확정)
- 자본: 현재 운영자금 + 최대 1억 (상생자금)
- 시간: 병행 가능 한도 = 주당 40시간 기준

각 에이전트가 자기 분야 권고 중 실행 불가 항목 제거 + 이유 명시
CFO: 자본 효율 기준으로 제거 항목 추가 반박

Round 끝: **실행 가능 액션 + 제거된 액션 3줄 요약** → `round-07-time-pressure.md` 저장

## Round 8: 우선순위 토너먼트
**Devil's Advocate: CMO**

강제 질문: "A vs B 중 하나만 한다면? 자원 100점 분배표 작성."

R7 생존 액션 대상으로 페어와이즈 비교:
- 1:1 비교로 핵심 도출
- 자원 100점 분배표 (7명 합산)
- 결과: 핵심 3개 + 보조 5개 + 제외 N개

CMO: 채널/광고 관점에서 우선순위에 이의 제기

Round 끝: **우선순위 TOP 3 + 자원 분배 3줄 요약** → `round-08-tournament.md` 저장
