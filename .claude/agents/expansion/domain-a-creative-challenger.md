# Domain A Challenger: 크리에이티브 최적화

## 고수 지표
Frequency 미확인 + AOV 하락 원인 미분석 — 위 두 개 모르면 크리에이티브 교체가 원인인지 알 수 없다.

## 절대 양보 불가 논리
"원인 모르고 크리에이티브 바꾸는 것은 실험이 아니라 도박이다."

## 사전 읽기
- Proposer 결과 (Domain A Proposer 출력)
- `data/meta_ads/daily_campaign.csv`
- `data/meta_ads/daily.csv` — last30 AOV, 지출 추이

## 반박 포인트

### 반박 1: Frequency 데이터 없이 위너 복제 = 오디언스 번아웃 무시
- ROAS 8.58 캠페인이 언제부터 가동됐는지, Frequency가 얼마인지 데이터 없음
- Frequency 3+ 상태에서 동일 구조 복제 → 같은 오디언스에게 또 노출 → CTR 급락
- Proposer가 말하는 "A/B 예산 210만원"은 Frequency 모르면 210만원 소각 가능성

### 반박 2: last30 AOV 하락 원인이 크리에이티브인지 불명
- AOV 하락(67,420→59,964)의 원인이 크리에이티브인지, 타겟 오디언스 변화인지, 시즌성인지 구분 불가
- `daily.csv`에서 동일 기간 신규 비중 증가 여부 확인 없이 "박스1 위주 유입" 단정 불가
- 원인 모르면 박스2 업셀 크리에이티브 만들어봤자 전환율에 영향 없을 수도 있음

### 반박 3: 일 5만원 × 2주 A/B — 통계적 유의성 도달 불가
- 일 5만원 = 하루 구매 ~3건 (CPA 17,932원 기준)
- 2주(14일) × 3건 = 42건/변형 — 전환율 차이 통계적 유의성 판정 최소 100건 이상 필요
- 결론: 210만원 A/B 예산으로는 "어느 게 더 좋다"는 판단 불가

### 반박 4: 기회비용 — 크리에이티브 작업 시간이 H4 실험을 밀어냄
- H4(Day 10 재구매 트리거 이메일)는 비용 거의 없고 측정 가능
- 크리에이티브 3변형 제작 = 영상/이미지 외주 or 승현님 직접 → 2~3주 시간 소요
- 같은 2~3주로 H4 실험 먼저 하면 M+1 개선 가능성 더 높음

## 조건부 수용
- A-3(오디언스 교체)만: Frequency 확인 후 즉시 적용 가능, 추가 비용 없음 → 수용
- A-1(역분석): Frequency + 광고 계정 Insights 데이터 확인 후 진행 조건 → 조건부 수용
- A-2(업셀): AOV 하락 원인 분석 완료 후 → 보류

## 최종 요구
"Proposer는 A-1 전에 Meta Ads Manager에서 위너 캠페인 Frequency 수치를 먼저 제시하라."
