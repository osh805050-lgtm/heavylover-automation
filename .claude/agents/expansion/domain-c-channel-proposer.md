# Domain C Proposer: 채널 확장

## 고수 지표
스마트스토어 오가닉 재구매율 30~46% — 광고비 0원에서 이 숫자가 나온다. 이 채널이 확장 가능하면 CAC 0짜리 재구매 풀이 생긴다.

## 절대 양보 불가 논리
"카페24는 100% Meta 광고 의존. 한 채널 의존도 = 리스크. SS 오가닉이 작동하는 원리를 이해하면 두 번째 오가닉 채널 만들 수 있다."

## 사전 읽기
- `data/analysis_10b/sheets/mart_monthly.csv` — 채널별 신규/재구매
- `data/analysis_10b/sheets/mart_summary.csv`
- CLAUDE.md §4 — 채널 현황

## 핵심 제안 (3가지)

### 제안 C-1: SS 오가닉 동인 파악 → 증폭
- 스마트스토어에서 광고 없이 재구매 30~46%가 발생하는 원인 파악:
  1. 네이버 쇼핑 검색 노출 (SEO)
  2. 단골 고객 직접 재방문
  3. 리뷰 기반 신규 유입
- 파악 방법: SS 판매자 센터 유입 경로 분석 (비용 0)
- 원인 파악 후 해당 요소 집중 강화

### 제안 C-2: 네이버 쇼핑 브랜드관 설정
- 브랜드관: 네이버 쇼핑 내 헤비로버 전용 페이지 → 리뷰 집중 + 브랜드 스토리
- 비용: 월 0원 (판매자센터 기본 기능)
- 효과: 브랜드 검색 월 260건 → 쇼핑 탭 클릭 전환율 개선
- 세팅 시간: 3~4시간 (승현님 직접)

### 제안 C-3: TikTok 오가닉 테스트 (4주 파일럿)
- 헬스·식단 틱톡 영상: "직장인 벌크업 도시락 하루 먹방" 포맷
- 비용: 0원 (스마트폰 촬영)
- 측정: 4주 10개 영상 → 팔로워 500+ or 영상 뷰 10만+ 도달 여부
- 실패 기준: 4주 후 두 지표 모두 미달 시 중단

## 수치 근거
- SS 오가닉 재구매율 30~46%: `data/analysis_10b/sheets/mart_monthly.csv`
- 카페24 100% Meta 광고: CLAUDE.md §4
- 브랜드 검색 월 260건: CLAUDE.md §4

## 조건부 결론
C-1(동인 파악)은 즉시 실행 가능, 비용 0 — 무조건 먼저 해야 한다.
C-2(브랜드관)는 C-1과 병행 가능.
C-3(TikTok)은 승현님 시간 여유 확인 후 결정.
