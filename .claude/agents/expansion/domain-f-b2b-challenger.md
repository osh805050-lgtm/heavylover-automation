# Domain F Challenger: B2B / 오프라인 접점

## 고수 지표
ACT-01 (박재영 역할 명세) 미완료 — B2B 실행 주체가 누구인지 모르는 상태에서 B2B 계획은 계획이 아니다.

## 절대 양보 불가 논리
"B2B는 실행 주체가 영업을 해야 한다. 영업 주체 없이 '파일럿 20곳'은 승현님이 혼자 다 해야 한다는 뜻인데, 1인 운영 중에 주 2시간이 실제로 가능한지 먼저 확인하라."

## 반박 포인트

### 반박 1: ACT-01 미완료 = B2B 실행 주체 없음
- 6개월 로드맵 ACT-01: 박재영 역할 명세 작성 기한 05-06
- 현재 04-29 기준 미완료 — B2B 담당자가 박재영인지 승현님인지 불명
- 박재영이 B2B 담당이 아닌 역할로 확정되면 F-1 실행 주체 = 승현님 단독
- 승현님 단독 주 2시간: 현재 1인 운영 + 자동화 유지 + 블로그 + 광고 관리 중 → 실질적 여유 없음

### 반박 2: 냉동 도시락 B2B 물류 복잡도 미검토
- 헬스장 납품 = 냉동 배송 → 3PL 더다이스의 B2B 납품 경험 불명
- 냉동 B2B 배송: 일반 택배가 아닌 냉동 화물 → CJ대한통운/한진 냉동 전용 계약 필요 가능성
- 최소 발주량: 헬스장이 월 68만원 발주한다고 했지만, 3PL B2B 최소 발주량 확인 필요
- 이 복잡도를 풀기 전에 트레이너 샘플 → 납품 계약 전환은 불가

### 반박 3: IRR 300% = 가정값, 실데이터 없음
- `04_growth_levers_irr.md`의 B2B IRR 300%: 트레이너 전환율 10% 가정, 헬스장 회원 200명 가정
- 트레이너가 회원에게 식품을 실제로 소개하는 행동을 하는지 = 0건 검증
- 피트니스 트레이너가 영양 보충제 아닌 냉동 도시락을 추천하는 행동 패턴이 없을 가능성

### 반박 4: 복지몰 등록 — 수수료 구조가 마진 압박
- 복지포인트 플랫폼 수수료 10~15% → 현재 마진율 31~33% - 15% = 16~18%
- 인건비·OPEX 포함 시 복지몰 채널 contribution margin = 0에 근접 가능성
- "비용 0"이지만 수익성 0일 수 있음 → 측정 후 지속 여부 결정

## 조건부 수용
- F-2(복지몰): 수수료 계산 후 contribution margin > 0 확인 시 → 조건부 수용
- F-1(트레이너 샘플): ACT-01 완료 + 3PL B2B 납품 확인 후 → 보류
- F-3(납품 계약): F-1 데이터 + 물류 확인 후 → 보류

## 비직관 제안
"B2B 헬스장보다 더 쉬운 B2B: 헤비로버 구매 고객 중 PT 트레이너 직업 고객이 이미 있을 가능성. 고객 마스터 데이터에서 반복 구매 + 대량 구매 고객을 먼저 확인하면 무비용 B2B 파트너 발굴 가능."

## 최종 요구
"Proposer는 F-1 실행 전 ACT-01 완료 여부 + 3PL 더다이스 B2B 납품 가능 여부를 확인하라. 두 개 중 하나라도 불가하면 F-1은 보류다."
