# Domain C Challenger: 채널 확장

## 고수 지표
1인 운영 병목 — 현재 카페24+SS 2채널도 자동화로 겨우 돌아간다. 채널 추가 = 운영 병목 악화.

## 절대 양보 불가 논리
"채널 추가 전에 기존 채널에서 뭘 놓치고 있는지부터 찾아라. SS 오가닉이 왜 작동하는지 모르면 증폭은커녕 깨뜨릴 수도 있다."

## 사전 읽기
- Proposer 결과 (Domain C Proposer 출력)
- `data/analysis_10b/sheets/mart_monthly.csv`
- CLAUDE.md §5 — 자동화 현황

## 반박 포인트

### 반박 1: SS 오가닉 동인 데이터가 실제로 있는지 확인 필요
- 네이버 스마트스토어 판매자센터 유입 경로 분석: 기능 있지만 검색어·유입경로 데이터가 충분히 세분화되는지 불명
- "비용 0, 즉시 가능"이라고 했지만 — SS 데이터 접근 권한·API 연동 여부 확인 필요
- 오가닉 재구매 30~46%가 "단골 직접 재방문"이라면 증폭 방법 없음

### 반박 2: 네이버 쇼핑 브랜드관 — 운영 시간 과소평가
- 브랜드관 초기 설정 3~4시간 맞지만, 이후 상품 등록·리뷰 관리·이벤트 업데이트 = 월 5~10시간
- 현재 SS는 자동화로 운영 중이지만 브랜드관은 콘텐츠 업데이트 수동 필요
- 승현님 1인 운영 상태에서 추가 5~10시간/월은 다른 작업을 밀어냄

### 반박 3: TikTok — 4주 10개 영상의 기회비용
- 영상 1개 제작 = 최소 1~2시간 (촬영+편집) → 4주 10개 = 10~20시간
- 같은 10~20시간으로: H4 실험 설계+실행, B-3 태깅 인프라 구축, 위너 캠페인 Frequency 분석 가능
- TikTok → Cafe24 전환 추적 구조 없음 (UTM 세팅 없으면 매출 기여 측정 불가)

### 반박 4: 채널 분산 vs 집중 — 지금 단계에서 분산의 근거
- 월매출 3,800만, 흑자선 5,000만 — 지금은 분산이 아니라 집중이 필요한 단계
- 카페24 ROAS 3.51이 Kill Criteria K1(2.8)에 70bp 여유만 있는 상태에서 새 채널 실험 여력 불명
- Proposer가 "SS 오가닉 작동 원리 이해 → 두 번째 오가닉 채널"이라고 했지만: SS 동인 파악에만 1~2개월 소요 가능

## 조건부 수용
- C-1(SS 동인 파악): 판매자센터 유입 분석 — 비용 0, 2시간 내 → 수용. 단, "증폭 계획"은 데이터 확인 후
- C-2(브랜드관): 월 운영 시간 5시간 이하 확인 후 → 조건부 수용
- C-3(TikTok): H4 + B-3 인프라 완료 후 여유 시간 생기면 → 보류

## 최종 요구
"Proposer는 C-1에서 SS 판매자센터 유입 경로 분석이 실제로 어떤 데이터를 보여주는지 먼저 확인 후 증폭 계획 수립하라."
