# Domain E Challenger: 콘텐츠 / SEO

## 고수 지표
블로그 → 카페24 전환 추적 없음 — 지금 당장 블로그가 매출에 기여하는지 0인지 모른다. 이 상태에서 투자 확대는 근거 없음.

## 절대 양보 불가 논리
"SEO는 결과가 3~6개월 후에 나온다. 지금 흑자선 5,000만에서 3,800만인 상황에서 3~6개월 후를 기다리는 투자가 맞는 타이밍인가?"

## 사전 읽기
- Proposer 결과 (Domain E Proposer 출력)
- CLAUDE.md §7 — 블로그 정책
- `docs/context/blog.md`

## 반박 포인트

### 반박 1: 현재 블로그 발행이 매출로 이어지는지 데이터 없음
- Proposer도 인정: 전환 추적 없음
- 기존 블로그 글이 검색 유입을 만드는지, 만든다면 구매로 이어지는지 = 완전 블랙박스
- E-3 UTM 세팅 먼저 → 기존 글 성과 측정 → "투자할 만한지" 판단 후 E-1 확대가 맞는 순서
- Proposer 제안은 E-1(투자 확대)과 E-3(측정 인프라)를 "동시"라고 했지만: E-3 없이 E-1 먼저 하면 측정 불가 투자

### 반박 2: SEO 효과 3~6개월 지연 — 지금 타이밍 문제
- 네이버 C-Rank/D.I.A+: 새 콘텐츠가 상위 노출까지 최소 2~4개월
- 월매출 3,800만 → 5,000만 달성이 급한 지금, 6개월 후 효과를 기다릴 여유 불명
- 같은 시간에 H4 실험 + ROAS 최적화가 더 빠른 매출 기여

### 반박 3: 오픈채팅 — 관리 리스크와 브랜드 이미지
- 불만 고객 유입 시: 오픈채팅은 공개 채널, 부정 후기 공개 확산 가능
- "코어팬 125명"이라고 했지만 오픈채팅은 링크 공유로 불특정 다수 진입 가능
- 운영: 주 1회 메시지 5분이라고 했지만, 고객 질문·불만 응대 = 예측 불가 시간 소요

### 반박 4: 브랜드 검색량 260 → 800 목표 — 근거 없음
- 6개월 내 3배 증가 목표의 근거가 없음
- 동일 카테고리(헬스 식품 D2C) SEO 사례 데이터 없이 "가능하다"는 가정
- 검색량 증가가 전환으로 이어지는 비율 데이터도 없음

## 조건부 수용
- E-3(UTM 세팅) 먼저: 비용 0, 2~3시간 → 즉시 수용. 단, 이게 E-1 확대의 선결조건
- E-1 확대: E-3 데이터 2개월 후 → "블로그 유입 전환율 > 0.5%" 확인 시 수용
- E-2(오픈채팅): 비공개(초대전용) 설정 + 모더레이션 규칙 확립 후 → 조건부 수용

## 비직관 제안
"브랜드 검색량 키우는 것보다 현재 검색량 260명이 왜 구매 안 하는지 분석이 먼저다. 브랜드 검색 → 카페24 랜딩 → 구매 전환율을 E-3으로 먼저 측정하면 260명에서 더 짜낼 수 있다."

## 최종 요구
"Proposer는 E-3 UTM 세팅 완료 후 현재 블로그 기여 전환율을 2주 측정한 뒤 E-1 확대 계획을 재제출하라."
