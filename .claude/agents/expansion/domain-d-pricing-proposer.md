# Domain D Proposer: 가격·패키지 최적화

## 고수 지표
재구매 AOV 86,529 — 신규 AOV 67,420 대비 1.28배. 정착 고객은 자연스럽게 더 많이 산다. 이 경향을 신규 단계부터 유도할 수 있다.

## 절대 양보 불가 논리
"박스1(18,000원)과 박스6(126,000원) 사이 빈 구간이 있다. 이 구간을 채우는 게 AOV 상승과 전환율 동시 개선의 가장 빠른 경로다."

## 사전 읽기
- `data/analysis_10b/unit_economics.json` — AOV 분포, 재구매 패턴
- `data/analysis_10b/sheets/mart_summary.csv` — SKU별 판매 현황
- CLAUDE.md §3 — 제품 라인업

## 핵심 제안 (3가지)

### 제안 D-1: 박스2 업셀 랜딩 실험 (추가 비용 0)
- 현재 제품 페이지: 박스1~박스6 나열식 → 박스2를 "가장 많이 선택" 배지로 강조
- 카트 담기 시 팝업: "박스1 구매 고객의 68%가 두 번째 주문에서 박스2로 업그레이드합니다"
- 비용: 카페24 상품 페이지 수정 (Claude 작업 2시간)
- 측정: 박스2 비중 변화 (현재 비중 → 4주 후 비중)
- 목표: 박스1 구매 비중 -10% → 박스2 비중 +10% → AOV +8,000원 추정

### 제안 D-2: 중간 가격대 박스3 신설 (SKU 추가 없이)
- 현재: 박스1(2개,18,000원) → 박스2(4개,34,000원) → 박스4(8개,68,000원)
- 갭: 34,000원~68,000원 사이 공백 → 박스3(6개,49,000원) 신설
- SKU 추가 아님 — 기존 제품 수량 조합만 변경, OEM 추가 발주 없음
- 목표: "박스2 사고 싶은데 박스4는 부담" 고객 포획

### 제안 D-3: P50 10일 정기 구독 옵션 (카페24 구독 기능 활용)
- 10일 자동 재주문 → P50과 정확히 일치, 이탈 없는 자동 재주문
- 구독 할인 5% 제공 → AOV는 -5%지만 구매 빈도 +100% (월 3회 vs 현재 1.5회)
- LTV 계산: 현재 M+1 14% 구조 vs 구독 확정 구조 비교
- 단, 카페24 구독 기능 실제 작동 여부 확인 필요
