# Domain E: 콘텐츠 / SEO
날짜: 2026-04-29

## Proposer 제안

**현황**
- 블로그: 화/목 주 2회 발행 중, 전환 추적 없음
- 코어팬(4회+): 144명 실측
- 브랜드 검색량: 월 260건
- CAC: 17,932원 (Meta), 오가닉 = 0원

**E-1: 블로그 키워드 버킷 3개 전략**
- 정보성(벌크업/헬스 식단) / 비교(단백질 비교) / 브랜드 방어
- 주 2회 발행 유지, UTM 전수 적용

**E-2: 코어팬 144명 카카오톡 오픈채팅**
- 비공개, 주 30분 운영, 시리얼 프리오더 첫 활용

**E-3: UTM + GA4 연동 (선결)**
- Cafe24 GA4 연동 1시간, UTM 파라미터 표준화

## Challenger 반박

**반박 1**: 흑자선 갭 1,200만원 + 오가닉 3~6개월 지연 — 지금 타이밍 불일치. Proposer 스스로 "단기 매출 기여 없음" 인정. 같은 시간을 Meta 크리에이티브 최적화에 쓰면 즉시 ROAS 개선 가능.

**반박 2**: Cafe24 GA4 연동은 "1시간"이 아닌 "반나절 구현 + 2주 검증". purchase 이벤트 중복 버그 빈발, UTM 파라미터 채널 그룹 설정, 전환 지연 처리까지 포함하면 데이터 신뢰도 확보에 수주 소요.

**반박 3**: 카카오 오픈채팅 — 비공개여도 대화 캡처 유출 통제 불가. 불만 집결 공간 될 위험. 채널 선택 근거 없음 (네이버 카페, 인스타 클로즈드 그룹이 더 통제 가능).

**비직관 발견**: 기존 블로그 글을 Meta 광고 랜딩으로 전환 — 신규 글 SEO 대기 없이 즉시 신뢰도 있는 소재로 광고 집행 가능. 투스텝(정보 → 구매) 방식으로 결제 퍼널 마찰 우회 효과도 기대 가능.

## 오케스트레이터 판정

| 제안 | 판정 | 이유 |
|---|---|---|
| E-3 GA4+UTM 연동 | **즉시 실행** | 타이밍과 무관하게 전환 추적 없이 마케팅 효과 측정 불가. 반나절 투자. 오늘 해야 함 |
| E-1 키워드 SEO | **보류** | E-3 2주 데이터 후 블로그 기여 전환율 확인. 1% 이상이면 확대, 미만이면 E-1 투자 재검토 |
| E-2 오픈채팅 | **보류** | 채널을 카카오 오픈채팅 → 통제 가능한 대안으로 변경 후 재논의. 시리얼 출시(6월) 전 2주 내 채널 확정 |
| 블로그→광고 소재 전환 (비직관) | **생존** | Domain A의 A-1과 연계. 정보성 블로그 글을 Meta 랜딩으로 활용하는 실험 — 소재 제작 0원, 즉시 가능 |

**타이밍 판정**: E 도메인 전체의 핵심 약점은 "지금 흑자선 미달 상황에서 3~6개월 오가닉을 기다리는 게 맞나"라는 질문에 답을 못 함. E-3(인프라) + 블로그→광고 전환(즉시 매출 기여 가능성)만 생존. SEO 확대는 흑자 달성 후.
