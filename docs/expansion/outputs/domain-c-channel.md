# Domain C: 채널 확장
날짜: 2026-04-29

## Proposer 제안

**채널별 재구매 현황 (mart_monthly.csv 실측)**
- SS 재구매율 43.0% vs 카페24 21.7% (2026-04)
- SS AOV: 79,671원 / 카페24 AOV: 92,747원
- SS 재구매 매출: 677만원 / 카페24 재구매 매출: 751만원 (통합 재구매의 각 47%, 53%)

**C-1: SS 오가닉 동인 파악 (즉시, 0원)**
- 판매자센터 유입 분석 4~6시간
- 리뷰 50건 수동 분류, 오가닉 순위 확인

**C-2: 네이버 쇼핑 브랜드관 (상표 등록 후)**
- 비용 0, 브랜드관 독립 공간 확보

**C-3: TikTok 4주 파일럿 (월 4,500만 이후)**
- Meta 위너 소재 전용, 120~200만원, ROAS 1.5 기준

## Challenger 반박

**반박 1**: SS 재구매율 43%가 독립 오가닉인지 "Meta 광고 → 브랜드 인지 → 네이버 재검색" 지연 효과인지 불명. SS AOV가 카페24보다 낮다는 점(79,671 vs 92,747)이 "이미 브랜드 아는 고객이 편의로 네이버 쓰는" 패턴을 지지함.

**반박 2**: C-1 "4~6시간 동인 파악"은 과소평가. 판매자센터는 유입 경로 대분류만 제공 — "왜 오가닉이 강한지" 인과 파악에는 4~8주 트렌드 추적이 필요.

**반박 3**: 브랜드관 상표 등록 타임라인 — 특허청 평균 심사 12~15개월. 이미 출원 중이어도 2026년 내 등록 완료는 빠듯. 브랜드관 입점에 네이버 자체 거래액 심사도 별도 존재.

**비직관 발견**: Meta 광고 소재 1개를 SS URL로 연결하는 소규모 테스트 (월 50~100만원, 4주). SS 광고 유입 고객의 재구매율이 현재 43%와 같으면 채널 특성, 낮으면 기존 43%가 기편향임을 증명 — C-1보다 인과가 명확한 데이터를 4주 안에 획득.

## 오케스트레이터 판정

| 제안 | 판정 | 이유 |
|---|---|---|
| C-1 SS 동인 파악 | **생존** (범위 축소) | 목표를 "동인 파악"이 아닌 "유입 채널 비율 확인 + 가설 3개 설정"으로 축소. 4~6시간 내 가능한 것만 한다 |
| SS 분리 실험 (비직관) | **생존** | C-1과 병행. Meta → SS 연결 파일럿 50~100만원으로 인과 관계 검증 — 4주 후 "오가닉인지 지연효과인지" 판정 |
| C-2 브랜드관 | **보류** | 상표 등록 완료 시점 + 네이버 심사 기간 확인 후 재판정. 2026년 하반기로 밀릴 가능성 높음 |
| C-3 TikTok | **탈락** | 흑자선 미달 상태에서 1인 운영 병목 추가, Meta 최적화 주의 분산, 측정 인프라 없음 3가지 동시 해당 |

**비직관 판정**: SS 재구매율 43%의 인과 규명이 채널 전략의 핵심. "오가닉 vs Meta 지연"을 4주 50만원으로 확인 가능한데 이걸 안 하면 채널 확장 모든 결정이 가정 위에서 이루어짐.
