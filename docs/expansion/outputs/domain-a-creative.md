# Domain A: 크리에이티브 최적화
날짜: 2026-04-29

## Proposer 제안

**캠페인 데이터 실측 (daily_campaign.csv)**
- 위너(26.2.21 abo): ROAS 6.20, AOV 81,918원, 12일 운영, 34건
- 현행(26.3.25 abo): ROAS 3.64, AOV 60,375원, 34일 운영, 249건
- 26.4.2 스케일 abo: ROAS 3.02 (K1 기준선 2.8에 22bp 여유)
- 현행 캠페인 34일 중 ROAS 2.8 미만 일수: 11일(32%)

**A-1: 위너 역분석 크리에이티브 3변형**
- 핵심 가설: 위너 ROAS 6.20의 원인은 CTR이 아니라 AOV 81,918원 (현행 대비 +36%)
- 소재 구조 역분석 → 묶음 노출 방식 3변형 제작 (15~30만원 or 0원 자체제작)
- 기대: last30 ROAS 3.51 → 4.5+ / AOV 60,375 → 75,000+

**A-2: 박스2 업셀 카피**
- "2박스로 2주치" 메시지로 묶음 구매 유도
- 비용: 0~5만원, 7일 AOV 72,000 목표

**A-3: 오디언스 교체**
- 현행 ROAS 후반 하락 → Frequency 소진 가설
- 구매 이력자 제외 오디언스 + 인터레스트 테스트 70만원

## Challenger 반박

**반박 1**: 위너 12일/34건은 통계적 유의성 부족. 중단 사유(예산소진/성과하락)가 없다. ROAS 6.20이 초반 스파이크일 경우 복제 캠페인도 시간이 지나면 3.64로 수렴.

**반박 2**: A-2 업셀 카피가 결제→구매 전환율 49.85%를 악화시킬 수 있음. 가격 저항 증가 → 전환율 45% 이하 시 AOV 상승 효과를 상쇄. 트레이드오프 수치 시뮬레이션 없음.

**반박 3**: Frequency 실측치 없이 오디언스 교체 70만원 집행은 가설 베팅. 인터레스트 타겟은 Broad 확정(CLAUDE.md) 방향과 충돌 소지.

**비직관 발견**: AOV 하락 원인이 소재가 아니라 SKU mix 변화(박스1 단품 주문 비중 증가)일 수 있음. 주문 데이터에서 1박스 vs 2박스+ 비율 변화 확인 — 비용 0, 소요 1시간. 이것부터 하면 A-1~A-3 중 어느 게 맞는지 방향이 나온다.

## 오케스트레이터 판정

| 제안 | 판정 | 이유 |
|---|---|---|
| A-1 위너 역분석 | **생존** (조건부) | 광고관리자에서 26.2.21 소재 원본 확인 + 소규모(일 5만원) 2주 독립 테스트 선행 조건 |
| A-2 업셀 카피 | **생존** (조건부) | 전환율 롤백 트리거(44% 이하 48h → 원복) 설계 후 실행. 비용 0이므로 트리거만 있으면 리스크 제어 가능 |

| SKU mix 확인 (비직관) | **즉시 실행** | 비용 0, 1시간, A-1~A-3 방향 결정의 선결 데이터 |

**판정 이유**: A-1·A-2는 조건 충족 시 즉시 실행 가능. Kill Criteria K1(ROAS 2.8) 여유가 26.4.2 캠페인 기준 22bp밖에 없으므로 소재 실험 2주 기간 중 K1 모니터링 필수.

**조건 요약**:
- SKU mix 분석 → A-1/A-2 실행 방향 확정
- A-2: 결제 전환율 44% 롤백 트리거 설계 후 실행
- A-3: 광고관리자 Frequency 확인 후 재논의
