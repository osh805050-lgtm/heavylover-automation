# Domain F: B2B / 오프라인 접점
날짜: 2026-04-29

## Proposer 제안

**데이터 실측 (코호트_고객마스터.csv)**
- 10회 이상 고반복 고객: 13명
- 최상위: 24회 구매, 간격 8일 (개인 단위 초과 소비 가능성)
- 5회 이상 고객: 100명

**F-1: PT 트레이너 샘플 20곳 (비용 12~20만원)**
- 경기도 헬스장 PT 트레이너 샘플 제공 + 리퍼럴 코드
- 실행 주체: 박재영 (ACT-01 완료 조건) or 승현님 직접

**F-2: 기업 복지몰 등록 (즉시, 0원)**
- 베네피아·이지웰·복지넷 등록, 수수료 10~15%

**F-3: 헬스장 납품 계약 (F-1 후, 1,000~1,500만원)**

## Challenger 반박

**반박 1**: F-2 복지몰 수수료 15% + 냉동 배송비 3,500~4,500원 공제 시 순마진 재계산 필요. "비용 0" = 등록비 없음이지, 운영 수익성이 0이 아니라는 뜻이 아님. 실마진이 10% 미만으로 떨어질 수 있음.

**반박 2**: F-1 트레이너 가설 — 13명 고반복 고객이 트레이너임을 확인한 적 없음. 증거 0건. 샘플 20곳 발송 전에 13명 대상 1:1 짧은 인터뷰로 추천 경로 확인 (비용 0, 2일).

**반박 3**: IRR 300%에 영업 시간 기회비용·리드타임(2~3개월)·반품 조건·냉동 배송 인프라 확인이 빠짐. 실제 IRR은 100~150% 수준으로 낮아질 가능성.

**비직관 발견**: 고반복 13명을 직접 앰배서더로 전환. "지인 추천 시 다음 구매 1박스 무료" 쿠폰 → 추천인 태그로 B2B 가설 검증 가능. F-1 샘플 비용보다 적게 들고, 4~6주 만에 트레이너 채널 존재 여부를 데이터로 확인.

## 오케스트레이터 판정

| 제안 | 판정 | 이유 |
|---|---|---|
| 앰배서더 실험 (비직관) | **즉시 실행** | 비용 1박스 원가 × 전환 건수. 트레이너 가설 검증 + F-1 방향 결정 데이터를 4주에 획득 |
| F-2 복지몰 등록 | **생존** (조건부) | 냉동 배송비 포함 실마진 계산 후 contribution margin > 0 확인. 1곳 파일럿 계약서 조건 검토 후 나머지 등록 |
| F-1 트레이너 샘플 | **보류** | ACT-01 완료 + 앰배서더 실험 2주 결과 + 3PL B2B 납품 가능 여부 확인 후 재논의 |
| F-3 헬스장 납품 | **보류** | F-1 전환 데이터 + 상생자금 확정(K4) + 냉동 물류 인프라 확인 후 |

**ACT-01 연동**: F-1·F-3 모두 박재영 역할 명세(05-06 기한)에 블로킹됨. 앰배서더 실험과 F-2만 ACT-01 없이 진행 가능.
