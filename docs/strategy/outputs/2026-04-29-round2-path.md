# Round 2: 10억 경로
날짜: 2026-04-29 (재생성: 2026-05-02)

강제 비직관 질문: **"시리얼을 취소하면 어떻게 되나?"**

---

## 시리얼 취소 시나리오

| 항목 | 시리얼 진행 | 시리얼 취소 |
|---|---|---|
| 초도비용 절감 | -3,500~4,000만 | +3,500~4,000만 운영자금 |
| 6월 매출 기여 (추정) | +500~1,500만/월 | 0 |
| 도시락 교차구매 효과 | +구매빈도 2배 가설 | 가설 검증 불가 |
| 1인 운영 부담 | 도시락 + 시리얼 병행 | 도시락 단독 집중 |
| 10억 경로 의존도 | 레버C로 +3.6억 (낙관) | 레버 A·B에 100% 의존 |

---

## A. 마진 원리주의자

**시리얼 취소 권고 (조건부)**: 시리얼 마진율 30% 미만이면 즉시 취소. 도시락 31% 마진에 100% 집중 시 흑자선 1,200만 갭 메우는 데 4~6개월 단축 가능.

**10억 경로 (마진 우선)**: 흑자선 5,000만 → 6,500만(90일) → 8,300만(12개월). 광고 스케일 없이 결제 퍼널 + 재구매만으로. 연 7~8억 현실적, 10억은 시간 12~18개월.

---

## B. 신규 획득 극대화론자 — 시리얼 유지

**반박**: 시리얼 취소 = 레버C 봉쇄. 도시락 단독으로 10억 = 월 8,300만 = 현재 매출 2.2배. 광고비 4~6배 필요. ROAS 디케이 리스크 고스란.

**대안**: 시리얼은 도시락 광고의 "보조 SKU"로 포지셔닝 — 시리얼 단독 광고 0원, 도시락 광고에 끼워팔기. 초도비용만 회수하면 OK.

**10억 경로**: 광고 +50% (월 1,000만) 4주 스텝 테스트, ROAS 3.0 유지 시 +1.5억/년. 동시 위너 캠페인 30% 비중. 12개월 내 9~12억 가능.

---

## C. 구조 비관론자 — 시리얼 취소 강력 권고

**핵심 반박**:
- 6월 출시 임박 + 1인 운영 + ACT-01 미완 = 실행 가능성 낮음
- 시리얼 OEM은 나비야 아님 = 신규 OEM 관계, 캐파·납기·품질 리스크 3중
- K5(프리오더 500봉 미달) 발동 시 초도비용 3,500~4,000만 손실 + 광고비 회수 불가

**10억 경로 (비관)**: 10억 불가능. 4억 → 7억 안정화 후 12~18개월에 10억. 시리얼이 레버가 아닌 부담.

---

## D. 자본 배분가 — Stage-gate 적용

**판정**:
- 시리얼 출시 결정은 이미 매몰 비용 발생 (품목제조보고 완료, 패키지 발주 직전) → 취소 비용 vs 진행 비용 비교
- 진행: 3,500~4,000만 + 광고 300만/월 + K5 미달 시 0 효과
- 취소: 매몰 약 500~1,000만 회수 불가 + 6개월 다른 레버 집중

**Stage-gate**:
1. 프리오더 (4~5월): 500봉 미달 시 정식 출시 보류
2. 정식 출시 (6월): 첫달 매출 500만 미달 시 광고 보류
3. 3개월 (9월): 누적 매출 < 1,500만 시 SKU 단종 결정
