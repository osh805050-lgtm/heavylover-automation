# Round 1: 현재 진단
날짜: 2026-04-29 (재생성: 2026-05-02)

강제 비직관 질문: **"광고를 완전히 끄면 어떻게 되나?"**

오케스트레이터 사전 계산: 광고 OFF 잔존 매출 추정 1,000~2,100만 (R1 결과에서 도출. 합의 불가 — 채널 분리 데이터 부재)

---

## A. 마진 원리주의자 (Margin Fundamentalist)

**진단**: 1주문 순기여 검증 부족. AOV 67,420(신규) / 86,529(재구매), 마진 31%. 1주문 순기여 신규 ≈ 20,900원 / 재구매 ≈ 26,800원. CAC 17,717원 차감 후 신규 1주문에 +3,000원 수준 — 흑자 조건 빠듯.

**핵심 병목**:
- AOV 위너 81,918 vs 현행 60,375 = 격차 32% → 위너 캠페인이 사라진 후 마진 구조 악화
- 결제 퍼널 49.85% = 광고비 절반 소실 = 사실상 CAC 35,000원

**Kill Criteria**: AOV 58,000원 미만 2개월 + ROAS 2.8 미만 → 광고 전면 축소. 현재 AOV 59,964원, 여유 1,964원밖에 없음.

---

## B. 신규 획득 극대화론자 (Acquisition Maximalist) — A에 반박

**반박**: 마진 원리주의 관점은 정태적. ROAS 3.51 last30 = 업계 평균 2.5 대비 +40% — 시장 대비 압도적 우위. CAC 17,717원은 경쟁 식품 D2C 25,000원~ 대비 30% 우위.

**기회비용 강조**: 현재 광고비 690만원/월은 흑자선 갭 1,200만원 메우는 가장 빠른 경로. 광고 OFF 잔존 1,000~2,100만 추정은 이미 신규 획득 멈췄을 때의 디케이 경로. 광고 -50% 시 2~3개월 내 4억대 후퇴.

**제안**: 위너 캠페인(ROAS 6.20) 재활성화 70만원으로 평균 ROAS 4.0+ 회복 가능.

## C. 구조 비관론자 (Structural Pessimist) — A·B에 반박

**반박**:
- B의 "ROAS 3.51 우위"는 일평균 지출 167,000원 구간만 검증. 월 1,000만원+ 구간 데이터 0. 26.4.2 abo 선례에서 이미 ROAS 3.02로 저하.
- 시리얼 출시 시 OEM 캐파 확인 안 됨. M+1 9.8%(2026-03) → K2 경계.
- A의 "AOV 격차" 원인이 SKU mix 변화일 가능성 — 광고로 해결 못 할 수 있음.

**3대 위험**:
1. ROAS 디케이 + 광고 스케일 동시 진행 = 자본 소각
2. 시리얼 K5 미달 + 상생자금 K4 발동 동시 = 자력 흑자 불가
3. 1인 운영 병목 = 위너 발견·CRM·SS 실험 동시 진행 불가능

## D. 자본 배분가 (Capital Allocator) — A·B·C 모두 반박

**Stage-gate 원칙**:
- 흑자선 5,000만 미달 상태에서 "광고 스케일 vs 재구매 개선" 양쪽 다 베팅 금지
- 비용 0인 액션부터 순차 — 결제 퍼널 개선·K1 코드 확인이 1순위
- 70만원 위너 재활성화 결정도 SKU mix 데이터(비용 0, 1시간) 확인 후

**자본 배분 원칙**: 외부 자금 의존 위험 — 상생자금 6,000만(거의 확정)은 운영자금이 아닌 시리얼 초도비용 3,500~4,000만 + 버퍼로만 사용. 광고 스케일 자금은 월 순이익 500만에서만 충당.

## E. 정체성 도전자 (Identity Challenger) — A·B·C·D 전부에 반박

**CLAUDE.md 재논의 금지 위반 점검**:
- C의 "시리얼 K5 미달 시 50% 축소"는 문제없음 (Kill Criteria K4·K5 명시)
- B의 "위너 ROAS 6.20 재현 가능" = 사용자 희망 영합. 12일/34건 통계적 유의성 부족.
- A의 "1주문 순기여" 강조는 Bear 시나리오 보수 운영 옳지만 Base 시나리오 기여 액션 봉쇄
- D의 "외부 자금 의존" 비판은 이미 상생자금 6,000만 거의 확정인데 배제 가정으로 보수 편향

**박재영 체제 점검**: ACT-01 박재영 역할 명세 (05-06 기한)가 4개 도메인 액션을 블로킹 중. 결정 구조 문제 — 명세 없이 진행되는 의사결정은 추후 충돌 위험.
