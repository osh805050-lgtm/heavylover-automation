# Round 5: Kill Criteria 합의
날짜: 2026-04-29 (재생성: 2026-05-02)

강제 비직관 질문: **"10억 목표를 6억으로 낮추면 어떻게 되나?"**

---

## 10억 → 6억 목표 하향 시나리오

| 항목 | 10억 목표 | 6억 목표 |
|---|---|---|
| 월 매출 목표 | 8,300만 | 5,000만 |
| 흑자선까지 갭 | -3,300만 | 0 (흑자선 달성) |
| 광고비 필요 | 월 1,500~2,000만 | 월 600~800만 (현재 수준) |
| 시리얼 의존 | 레버C +3.6억 필수 | 시리얼 보조 (레버C 0~1.5억 OK) |
| 외부 자금 의존 | 1억 필수 | 6,000만(거의 확정)으로 충분 |
| 1인 운영 부담 | 한계 초과 | 박재영 명세 후 안정 |
| 12개월 후 자본 상태 | 0 (자본 소각 가능) | +5,000~8,000만 |

→ **10억 = 자본 소각 베팅 / 6억 = 자력 흑자 + 자본 적립 경로**.

---

## 5명 입장

### A. 마진 원리주의자
- "6억 = 흑자선 달성 + 자본 적립 가능. 10억은 단위경제학 검증 후 +12개월에 도전"
- 권고: **1차 목표 6억 (2026 연), 2차 목표 10억 (2027 상반기)**

### B. 신규 획득 극대화론자
- "6억 목표는 타겟 시장 동결. 대기업 진입 시 CAC 급등으로 6억도 위험"
- 권고: **10억 목표 유지. 단 외부 자금 + ROAS 검증 단계별**

### C. 구조 비관론자
- "현재 데이터로 10억 가능성 20%. 6억 = 현실 시나리오 45% 확률"
- 권고: **6억 명시 채택. 10억은 stretch goal**

### D. 자본 배분가
- "현금흐름 시뮬레이션 결과 10억 추구는 12개월 후 자본 잔고 0. 6억은 +5,000만 적립"
- 권고: **6억 → 8억 → 10억 단계 목표**

### E. 정체성 도전자
- "10억은 박재영 체제 활용 전제. 명세 확정 + 6개월 가동 후 재판정"
- 권고: **목표 결정 보류 — ACT-01 완료 후 수정**

3. **K8 현금흐름 즉시 모니터링 가동** (만장일치 — R9 빠진 관점 1순위)
4. **시리얼 Stage-gate 3단계** (R2 합의 유지)
