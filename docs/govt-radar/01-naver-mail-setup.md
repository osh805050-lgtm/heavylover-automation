# 네이버 메일 IMAP + 앱 비밀번호 발급 가이드

**왜 필요한가**: 정부 지원사업 알림 메일이 네이버 메일로 옵니다. 자동화 시스템이 이 메일들을 매일 읽어서 "혹시 1차 크롤링에서 빠진 공고가 있는지" 교차 검증하려면, 네이버 메일에 프로그램이 접속할 수 있게 해야 합니다.

**소요 시간**: 약 10분

**사전 준비**: 네이버 ID 비밀번호, 휴대폰

---

## 왜 일반 비밀번호로는 안 되는가

네이버는 보안상 **외부 프로그램이 일반 비밀번호로 메일 접속하는 걸 막아두었습니다.** 그래서 두 가지가 필요합니다:

1. **2단계 인증 켜기** — 로그인 시 휴대폰 인증 추가
2. **앱 비밀번호 발급** — 외부 프로그램용 16자리 전용 비밀번호

이 둘이 켜져야 자동화 시스템이 네이버 메일을 읽을 수 있습니다.

---

## STEP 1: IMAP 사용 설정

IMAP은 "외부 프로그램이 메일함을 읽을 수 있게 해주는 통로"입니다. 기본은 꺼져 있어요.

1. 네이버 로그인 → **메일** 들어가기 (mail.naver.com)
2. 좌측 하단 **환경설정** (톱니바퀴 아이콘) 클릭
3. 상단 탭에서 **POP3/IMAP 설정** 클릭
4. **IMAP/SMTP 설정** 탭 선택
5. **IMAP/SMTP 사용** → **사용함** 선택
6. 하단 **확인** 버튼

✅ **확인 포인트**: 페이지 하단에 IMAP 서버 정보가 보이면 성공
- IMAP 서버명: `imap.naver.com`
- IMAP 포트: `993`
- 보안 연결: SSL 사용

---

## STEP 2: 2단계 인증 켜기

⚠️ **이미 켜져 있으면 STEP 3으로 건너뛰세요.**

1. 네이버 우측 상단 프로필 → **내 정보** 클릭
2. 좌측 메뉴 **보안설정** 클릭
3. **2단계 인증** 항목 → **설정** 클릭
4. 휴대폰 번호 입력 → 인증번호 받기 → 입력
5. 인증 완료

✅ **확인 포인트**: 보안설정 페이지에 "2단계 인증: 사용 중"이 보임

---

## STEP 3: 애플리케이션 비밀번호 발급

이게 자동화 시스템에 넣을 진짜 비밀번호입니다. **일반 비밀번호와 다릅니다.**

1. 네이버 **내 정보** → **보안설정**
2. 하단 **애플리케이션 비밀번호** → **관리** 클릭
3. **애플리케이션 비밀번호 생성** 클릭
4. 이름 입력: `heavylover-govt-radar` (아무 이름이나 OK, 나중에 어디서 쓰는지 알아보려고 적는 것)
5. **생성** 클릭
6. **16자리 비밀번호가 화면에 표시됨** — 예: `abcd efgh ijkl mnop`

⚠️ **이 비밀번호는 한 번만 보여줍니다. 절대 닫기 전에 메모장에 복사해두세요.**

복사할 때 띄어쓰기는 빼도 되고 그대로 둬도 됩니다 (시스템에서 알아서 처리).
- 그대로: `abcd efgh ijkl mnop`
- 띄어쓰기 제거: `abcdefghijklmnop`

✅ **확인 포인트**: 메모장에 16자리 비밀번호가 저장됨

---

## STEP 4: 잘 됐는지 미리 테스트 (선택)

진짜 IMAP 접속이 되는지 미리 확인하고 싶으면:

### 방법 A: 네이버 메일 앱에서 테스트
스마트폰 기본 메일 앱 → 계정 추가 → 기타(IMAP) 선택 → 아래 정보 입력
- 이메일: `내아이디@naver.com`
- 비밀번호: **STEP 3에서 받은 16자리** (일반 비밀번호 ❌)
- 수신 서버: `imap.naver.com`, 포트 `993`, SSL 사용
- 송신 서버: `smtp.naver.com`, 포트 `465`, SSL 사용

메일이 동기화되면 성공.

### 방법 B: 그냥 다음 단계로
귀찮으면 STEP 5로 넘어가세요. 자동화 시스템 만들 때 어차피 테스트합니다.

---

## STEP 5: 메모해둘 정보

다음 단계(GitHub Secrets 등록)에서 이 두 가지가 필요합니다:

```
NAVER_MAIL_USER       = osh805050           ← @naver.com 앞부분만
NAVER_MAIL_APP_PASSWORD = abcdefghijklmnop  ← STEP 3에서 받은 16자리
```

⚠️ **NAVER_MAIL_USER는 이메일 앞부분만**입니다. `osh805050@naver.com` 전체가 아니라 `osh805050`만.

(승현님 실제 네이버 ID는 직접 확인하세요 — `osh805050@naver.com`이 맞는지)

---

## 자주 발생하는 문제

### "IMAP/SMTP 설정" 메뉴가 안 보여요
→ 네이버 메일 우측 상단 **PC버전** 으로 전환했는지 확인. 모바일 화면에서는 일부 메뉴가 숨겨집니다.

### 앱 비밀번호 발급 메뉴가 회색이에요
→ 2단계 인증이 안 켜져 있어서. STEP 2 다시 확인.

### 16자리 비밀번호를 잃어버렸어요
→ 다시 발급하면 됩니다 (기존 건 폐기 → 새로 생성). 이미 다른 곳에서 쓰고 있던 거라면 그쪽도 새 비번으로 교체해야 함.

### 보안 등급이 낮은 앱은 차단됩니다 메시지
→ 일반 비밀번호로 접속 시도해서 그래요. **반드시 앱 비밀번호** 사용.
