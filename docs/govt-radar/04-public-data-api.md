# 공공데이터포털 API 발급 가이드

**왜 필요한가**: 정부 사이트(K-Startup, 기업마당 등)는 직접 크롤링이 자주 깨집니다. 공식 API를 쓰면 영구 안정이고, 발급도 무료입니다.

**소요 시간**: 약 15~20분

**최종 결과**: K-Startup 공고가 매일 빠짐없이 자동 수집됨

---

## STEP 1: 공공데이터포털 가입

1. https://www.data.go.kr 접속
2. 우측 상단 **회원가입** 클릭
3. **일반회원**으로 진행
4. 이메일 인증 후 가입 완료

⚠️ **이미 계정 있으면 STEP 2로**.

---

## STEP 2: K-Startup API 검색

1. 메인 페이지 검색창에 **`K-Startup`** 입력 + 엔터
2. 검색 결과에서 다음 항목 찾기:
   - **창업진흥원_K-Startup 사업공고 정보**
   - 영문명: `kisedKstartupService`
3. 클릭해서 상세 페이지 들어가기

✅ **확인 포인트**: "OpenAPI" 라벨 + "활용신청" 버튼이 보임

---

## STEP 3: 활용신청

1. 상세 페이지에서 우측 **활용신청** 클릭
2. 다음 항목 입력:
   - **활용목적 선택**: `기타` 선택
   - **활용목적 상세**: 아래 텍스트 그대로 복사 붙여넣기
     ```
     자사 식품 D2C 브랜드(헤비로버)의 정부지원사업 모니터링 자동화 도구 개발.
     매일 K-Startup 공고를 수집해 자사 사업과 적합한 공고를 사내 알림으로
     필터링·전달하는 내부 운영 시스템에 활용.
     ```
   - **시스템 유형**: `일반` 또는 `웹사이트`
   - **라이선스 표시**: 동의 체크
3. **신청** 버튼

✅ **확인 포인트**: "신청 완료" 메시지 + "마이페이지에서 확인 가능"

---

## STEP 4: 인증키 받기 (자동승인 보통 1~5분)

1. 우측 상단 **마이페이지** 클릭
2. 좌측 메뉴 **오픈API** → **개발계정** 클릭
3. 방금 신청한 **창업진흥원_K-Startup 사업공고 정보** 클릭
4. 페이지 하단 **인증키** 영역 확인
   - **일반 인증키 (Encoding)**: `abc%2BDEF%2FghiJKL...` 같은 긴 문자열
   - **일반 인증키 (Decoding)**: `abc+DEF/ghiJKL...` 같은 긴 문자열
5. **Decoding** 키를 복사해두기 (메모장 등)

⚠️ 자동승인이 즉시 안 되면 1~5분 기다린 후 새로고침. 평일 오후라면 거의 즉시, 주말·공휴일이면 다음 영업일까지 걸릴 수 있음.

⚠️ Encoding/Decoding 둘 다 보이는데 **Decoding** 쓰세요. Python에서 자동으로 Encoding 처리.

---

## STEP 5: 테스트 — 진짜 작동하는지

마이페이지 → 해당 API 페이지에 **상세설명** 또는 **참고문서** 다운로드 버튼이 있어요. 거기에 샘플 URL이 들어있는데, 일단 건너뛰셔도 됩니다. 우리는 코드에서 테스트할게요.

발급 완료되면:
- 메모장에 인증키 복사 (Decoding 버전)
- 다음 단계로

---

## STEP 6: 인증키를 두 곳에 등록

### 6-1. 로컬 `.env` 파일

PowerShell에서:
```powershell
cd "C:\Users\osh80\OneDrive\바탕 화면\heavylover-automation"
notepad .env
```

맨 아래 한 줄 추가:
```
DATA_GO_KR_API_KEY=발급받은_Decoding_인증키_그대로
```

⚠️ 따옴표 ❌, 양옆 띄어쓰기 ❌

저장(Ctrl+S) → 닫기

### 6-2. GitHub Secrets

1. https://github.com/osh805050-lgtm/heavylover-automation/settings/secrets/actions
2. **새 저장소 비밀** (New repository secret)
3. 이름: `DATA_GO_KR_API_KEY`
4. 값: 같은 인증키
5. **추가**

---

## STEP 7: 알려주세요

발급 + 등록 완료되면 저한테 "발급 완료"만 알려주세요. 그러면 제가:
1. K-Startup API 클라이언트 코드 작성 (`lib/kstartup_api.py`)
2. 기존 govt_sources.py에 통합
3. 테스트 실행 → 진짜 공고 들어오는지 확인
4. GitHub Actions 워크플로에 인증키 추가

까지 진행합니다.

## (참고) 기업마당 API도 같이 발급 가능

같은 사이트에서 **`기업마당`** 검색하면 비슷한 API들이 있습니다:
- **중소벤처기업부_기업마당 사업공고**
- 절차 동일. 한 번에 같이 신청해두면 좋음.

이건 K-Startup 끝낸 후 여유 있을 때 추가하셔도 됩니다 (지금은 메일이 커버 중).

## 자주 발생하는 문제

### "활용신청" 버튼이 회색이에요
→ 로그인 안 됨. 우측 상단 로그인 확인.

### 인증키가 안 나와요
→ 자동승인 대기 중일 수 있음. 마이페이지 새로고침. 1시간 지나도 안 나오면 평일 영업시간 기다리기.

### 트래픽 제한 있나요?
→ 일 1,000건 무료. 우리는 하루 한 번 ~30건만 호출하니 여유 충분.

### 운영계정으로 전환 필요해요?
→ 처음엔 개발계정으로 충분. 일 1,000건 넘기면 운영계정 전환. 우리는 안 넘김.
