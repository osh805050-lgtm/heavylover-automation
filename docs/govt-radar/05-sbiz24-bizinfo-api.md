# 소상공인24 + 기업마당 API 발급 가이드

**왜 필요한가**: 소상공인24는 직접 크롤링이 안 되고(SPA), 기업마당도 메일로만 들어오고 있습니다. 공식 API를 쓰면 진짜 수집·정확도가 올라갑니다.

**소요 시간**: 약 10~15분 (한 번에 두 개 신청)

**최종 결과**: 소진공·중기부 공고가 매일 빠짐없이 자동 수집됨

---

## 시작 전 — 인증키는 그대로 사용

**이미 발급받은 K-Startup API 인증키 (`DATA_GO_KR_API_KEY`)를 그대로 씁니다.**
data.go.kr 계정 하나로 여러 API를 쓸 수 있어요. 활용신청만 추가하면 됩니다.

---

## STEP 1: 소상공인24 API 활용신청

### 1-1. 검색
1. https://www.data.go.kr 접속 + 로그인
2. 메인 검색창에 **`소상공인시장진흥공단`** 입력 + 엔터
3. 결과 화면에서 좌측 필터 **OpenAPI** 체크 (그냥 데이터 파일 말고 API만 보기)

### 1-2. 추천 API들 (상위 노출되는 것 위주, 다 신청해도 됨)
다음 중 보이는 것 **모두 활용신청** (한 번에 여러 개 가능):

| API 이름 | 우선순위 |
|---|---|
| **소상공인시장진흥공단_지원사업안내** | ⭐⭐⭐ 핵심 |
| **소상공인시장진흥공단_정책자금 정보** | ⭐⭐ |
| **소상공인시장진흥공단_교육일정** | ⭐ |
| **소상공인 정책자금 융자대상 업종** | 참고용 |

⚠️ **이름이 정확히 일치 안 해도 됩니다.** "소상공인" 검색 결과에서 **공고/지원사업/정책자금** 들어간 OpenAPI는 다 신청.

### 1-3. 활용신청
각 API 클릭 → 우측 **활용신청** → 다음 텍스트 그대로 복붙:

```
자사 식품 D2C 브랜드(헤비로버)의 정부지원사업 모니터링 자동화 도구 개발.
매일 소상공인시장진흥공단 지원사업 공고를 수집해 자사 사업과 적합한 공고를
사내 알림으로 필터링·전달하는 내부 운영 시스템에 활용.
```

→ **신청** 클릭

---

## STEP 2: 기업마당(중기부) API 활용신청

### 2-1. 검색
같은 사이트 검색창에 **`기업마당`** 또는 **`중소벤처기업부`** 입력

### 2-2. 추천 API들
| API 이름 | 우선순위 |
|---|---|
| **중소벤처기업부_기업마당 사업공고** | ⭐⭐⭐ 핵심 |
| **중소벤처기업부_정책자금 정보** | ⭐⭐ |
| **중소벤처기업진흥공단_사업공고** | ⭐⭐ |

⚠️ "기업마당" 검색 결과에 OpenAPI가 안 보이면 **"중소벤처기업부"**로 다시 검색.
⚠️ 활용신청 텍스트는 STEP 1과 같은 거 사용 ("소상공인시장진흥공단" → "중소벤처기업부"로 한 단어만 바꿔서)

---

## STEP 3: 자동승인 대기 (보통 1~5분)

1. 우측 상단 **마이페이지** → **오픈API** → **개발계정**
2. 신청한 API 목록에 **승인** 또는 **활용중** 표시 확인
3. 승인 안 됐으면 새로고침. 1시간 지나도 안 되면 평일 영업시간 기다리기

---

## STEP 4: 알려주세요

승인된 API **이름 목록**만 알려주세요. 정확한 엔드포인트 URL은 제가 마이페이지 화면 정보로 알아낼 수 있어요.

예시 답변:
> "승인 완료. 신청한 거:
> - 소상공인시장진흥공단_지원사업안내
> - 중소벤처기업부_기업마당 사업공고"

---

## STEP 5 (저쪽 작업): 코드 작성

승현님이 STEP 4 알려주시면 제가:
1. 마이페이지에서 각 API 엔드포인트·파라미터 확인
2. `lib/sbiz24_api.py`, `lib/bizinfo_api.py` 작성
3. K-Startup과 동일 패턴으로 통합
4. 테스트 → 진짜 들어오는지 확인
5. GitHub Actions에 동일 키 사용 (Secrets 추가 작업 X — 기존 `DATA_GO_KR_API_KEY` 재사용)

## 자주 발생하는 문제

### "이미 활용중" 표시
→ 이미 신청했었음. 그대로 사용 OK.

### 검색해도 OpenAPI가 안 보임
→ 좌측 필터 "오픈API" 체크 확인. 또는 데이터 종류 필터를 "전체"로.

### 활용신청 버튼이 회색
→ 로그인 안 됨. 우측 상단 로그인 상태 확인.

### 자동승인이 너무 오래 걸림
→ data.go.kr는 평일 영업시간(9~18시)에 검토. 주말·공휴일 신청은 다음 영업일까지 걸림.
→ 단, **K-Startup 같은 인기 API는 즉시 자동승인**, 덜 알려진 API는 수동 검토 들어갈 수 있음.
