## 2. 가설 3개

### 가설 A: 소규모 + 고의도 유입 (2026-04-30 채택)
- 2026-02 광고 지출 352,422원 (전체 평균 대비 -85%)
- ROAS 4.70 (해당월 가장 높음)
- 광고 7주 중단 기간(2026-01-05~02-20)에 검색·재방문 등 고의도 유입만 진입
- 신규 88명 = 소규모 → 자연 도달한 고의도 사용자만 유입
- **재현 불가능**: 대규모 광고 유입 시 평균치(M+1 12.8%)로 회귀

### 가설 B: 설날 시즌성
- 2026-02 설날 직후 (2026-02-09~12)
- M+2(10.2%)도 평균(7~8%)보다 높음 → 시즌성 부분 기여
- 설명력: 30% 수준

### 가설 C: 2025-12 코호트 M+2 혼입
- 2025-12 코호트(390명, M+1 16.9%)의 M+2가 2026-02 통계에 잘못 합산됐을 가능성
- 데이터 검증 결과: 코호트 분리 정상. 가설 기각

## 4. CRM 설계에 미치는 영향

**잘못된 설계**: 2026-02 코호트 M+1 29.5%를 목표로 잡고 D+10 알림톡 설계
- → 비용·기대치 인플레이션
- → 알림톡 효과 미달 시 잘못된 회수 결정 (실제로는 12.8% 기준선이 정답)

**올바른 설계**: 12.8% 기준선 + 2~3%p 개선 목표
- D+10 알림톡: 12.8% → 15~16% 목표
- 2026-02 인사이트는 "고의도 유입 채널 확대"로 재해석
  - 네이버 SEO 블로그 (E-3 우선)
  - 자사몰 직접 방문 유도
  - 13명 코어팬 레퍼럴
