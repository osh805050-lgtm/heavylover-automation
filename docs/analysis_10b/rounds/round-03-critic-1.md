## 치명적 가정 TOP 3

### 가정 1: ROAS 3.5가 광고비 4~8배 확대 시에도 유지된다
- **틀리면**: Base 시나리오 레버 A +4.8억이 6분의 1 수준(8,160만)으로 붕괴
- **근거**: 현재 일 최대 지출 346,202원 수준에서만 ROAS 3.51 검증됨. 월 1,000만원+ 구간 데이터 없음. 26.4.2 abo 선례에서 이미 3.02로 저하.
- **검증**: 월 1,000만→2,000만→3,000만원 4주 스텝 테스트. 비용: 약 6,000만원, 3개월.
- **위험도**: 높음

### 가정 2: M+1을 16~20%로 개선하는 메커니즘이 존재한다
- **틀리면**: LTV 31,687원 하락(실측 p_2_to_3이 assumed 35%보다 낮으면 LTV 최대 25,000원 수준), LTV/CAC 1.79 → 1.4로 하락
- **근거**: p_2_to_3(35%), p_3_to_4(40%) 모두 "assumed". 실측 코호트에서 n≥100 기준 M+1 14% 이상 안정 달성 달은 2025-12(16.9%) 단 1개월뿐.
- **검증**: 카페24 3.4회 구매 분포로 p_2_to_3 즉시 역산 가능. 비용 0원, 1일.
- **위험도**: 높음 (LTV 계산 전체 오염)

### 가정 3: 상생자금 1억이 확보된다
- **틀리면**: Bear 시나리오(8.88억)로 후퇴. 10억 달성 시한 1~2년 지연.
- **근거**: "현장실사 대기" 상태. K4(탈락 시 시리얼 50% 축소) 이미 설정됨.
- **검증**: 외부 변수. 대응 플랜 B(탈락 시 레버 B+C만으로 Bear 달성 가능 여부) 사전 수립. 비용 0원, 반나절.
- **위험도**: 중간~높음

## 실패 시나리오 3개

1. **ROAS 계단식 붕괴**: 월 1,000만원+ 구간에서 타겟 포화 + 소재 피로 동시. ROAS 2.5 하락 시 실질 영업이익 -500~800만원/월.
2. **시리얼 K5 발동 + OEM 위약금**: 최소 발주 위약금 500만~2,000만원 가능성. Bear 시나리오 하단 이탈.
3. **CPA 상승 + 결제 퍼널 미개선 동시 발생**: CPA 30,000원 돌파 시 K7(LTV/CAC < 1.5) 근접. 광고 손익분기 위협.
