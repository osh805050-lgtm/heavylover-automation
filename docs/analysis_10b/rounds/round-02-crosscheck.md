1. **OEM-퍼널 연계 리스크**: 시리얼 6월 출시 + 도시락 병행 시 나비야 캐파 충돌 → 퍼널 개선 전 캐파 확인 필수

4. **Cafe24 CRM API 스펙 미확인**: 그룹 분류 API 엔드포인트 존재 여부 미확인 — "이번 주 구축" 일정이 사전 검증 없이 설정됨
