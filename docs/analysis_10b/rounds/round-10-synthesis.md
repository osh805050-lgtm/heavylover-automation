## 1. 10억 달성 가능성 평가

| 시나리오 | 확률 | 조건 | 예상 연매출 |
|---|---|---|---|
| 낙관 | 20% | ROAS 스케일 유지(월 1,000만+) + M+1 16~20% + 상생자금 1억 + 시리얼 K5 통과 | 12~15억 |
| 현실 | 45% | ROAS 3.0~3.5 유지 + M+1 14~16% + 상생자금 탈락 + 시리얼 점진 성장 | 7~9억 |
| 비관 | 35% | ROAS 디케이(< 2.8) + M+1 12% 고착 + 시리얼 K5 미달 | 4~6억 |

**핵심 판단 근거**:
- R3 비판: 낙관 조건 4개 중 현재 검증된 것 0개. "10억 현실" 이 아닌 "10억 가능" 수준.
- R5 스트레스: ROAS -30% 시 6개월 손실 7,792만원 → 비관 방어가 낙관 추구보다 먼저.
- R6 가정 역전: 현재 연매출 페이스 4.56억에서 10억은 2.2배 성장 필요 — M+1 12.8% + ROAS 디케이 + 1인 운영 병목 세 가지가 동시에 해결되어야 가능.
- 현실 시나리오(45%)가 가장 높은 이유: 결제 퍼널 + K1 수정만으로도 7~9억은 수학적으로 달성 가능하나, 10억은 외부 자금이나 ROAS 스케일 검증 없이 단독 불가.

### 7명 합의 결론 (만장일치)

1. K1 자동 중지 코드 확인+수정이 모든 것보다 먼저 (1~2시간, 비용 0)
2. 결제 퍼널 49.85% = 최대 병목. 단, 원인 진단 없이 처방만 집행하면 효과 불확실
3. 2026-02 코호트 분석이 CRM 설계 전 선행 필수 (0원, 2시간)
4. LTV 31,687원의 p_2_to_3·p_3_to_4 "assumed" 수치 → 이번 주 실측 역산 필수
5. OEM 나비야 캐파 확인 없이 스케일업 계획 수립 불가

### 7명 최종 불합의 (영구 박제)

- "상생자금 탈락 시 즉시 전면 중단 vs 50% 축소": CFO(전면 중단) vs 성장해커(50% 축소). 사용자 판단 필요.
- "위너 캠페인 재현 목표 설정": CMO(현행+10~20% 현실 목표) vs 성장해커(몰빵 시 ROAS 5.0+ 수학적 가능). 실험으로만 해결.
- "M+1 이탈 원인": 고객인사이트(잊어서) vs R4 Devil's Advocate(필요 없어서). 코호트 분석 후 재논의.

### 핵심 3개 레버 (자원 집중 우선순위)

| 순위 | 레버 | 예상 효과 | 조건 |
|---|---|---|---|
| 1 | 결제 퍼널 개선 | 월 +240~354만원 (추정) | 원인 관측 병행 |
| 2 | K1 자동 중지 수정 | 최대 손실 차단 7,792만원 | 코드 확인 선행 |
| 3 | D+10 알림톡 자동화 | M+1 +3~5%p (추정) | 코호트 분석 완료 후 |

### 비직관 발견 TOP 3 (R1~R9에서 도출)

1. **광고 AOV(62,019원) < 재구매 AOV(86,529원) 격차**가 ROAS 격차보다 중요. 위너 캠페인이 높은 이유는 소재가 아닌 AOV를 끌어올리는 타겟 구조일 가능성. *(62,019원 = 26.3.25 abo 캠페인 AOV; 카페24 전체 last30 AOV는 59,964원 — R1 출처)*
2. **SS 재구매율 43% = 오가닉이 아닐 수 있음**. Meta 광고 지연 효과일 경우, 채널 다각화 전략 전체 재검토 필요. 50만원 4주 실험으로 확인 가능한데 아직 미착수.
3. **2026-02 코호트 M+1 29.5% 이상치**는 CRM 목표 기준이 아닌 "소규모 고의도 유입" 실험의 증거. 대규모 유입 시 재현 불가. CRM 목표는 12.8% 기준 설계해야 한다 (과대 목표 차단).
