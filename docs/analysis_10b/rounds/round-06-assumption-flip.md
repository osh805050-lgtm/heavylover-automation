## "360M 천장" 이유 3가지

1. **재구매 구조 고착**: M+1 12.8% + 3.4회 구매 후 이탈 패턴이 유지되면 신규-이탈 균형 수렴
2. **ROAS 진동 구간**: 광고비 증가 → ROAS 2.8 구간 → K1 발동 → 축소 → 반복. 광고 스케일 원천 차단
3. **1인 운영 실행 병목**: 23개 액션 중 13개가 이번 달 착수 가능이나 오승현 1인 실행 속도 제한
