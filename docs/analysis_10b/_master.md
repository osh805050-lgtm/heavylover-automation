## §4. IC 8섹션 진단

### §4.1 시장 (Market)
- 한국 HMR 7조원 (2026, 유로모니터)
- 단백질 식품 CAGR 30%+
- 운동 직장인 벌크업 세그먼트: 추정 10~15만명 → 1만명×연 10만원 필요 (10억 기준)

### §4.2 제품 (Product)
- 1인분 800kcal/40g 포지셔닝 — 직접 경쟁자 없음 (추정, 6~12개월 지속)
- IQF 급속냉동 + 수비드 + HACCP — 기술 우위
- **HACCP 해자 허점**: 나비야 OEM 보유. 헤비로버 자체 인증 아님. 독점 계약 미확인

### §4.3 채널 (Channel)
- 카페24 자사몰 ~57% / SS ~30% / 기타 ~13%
- SS 재구매율 43% — Meta 지연효과 가능성 미검증
- **카페24 SPOF**: 이전 시 매출 6개월 후퇴

### §4.4 단위경제학 → §1 참조

### §4.5 운영 (Ops)
- 11시 발주, 송장 등록, OAuth 갱신, 09시 재구매 리포트 — 자동화 95%+
- 카페24 N10→N20, SS 신규→발주확인 — 수동 (API 한계)
- **OEM 캐파**: 나비야 미확인. 시리얼 신규 OEM 미검증

### §4.6 자본 (Capital)
- 월 고정비 500~550만 (박재영 100만 포함)
- 4월 순이익 실측 500만
- 흑자선 5,000만 갭 1,200만
- 상생자금 6,000만 거의 확정 → 시리얼 초도비 + 운영 버퍼

### §4.7 인력 (People)
- 오승현 1인 + 박재영 (역할 미확정)
- ACT-01 박재영 명세 (05-06) — 4도메인 액션 블로킹 중
- 1인 운영 한계: 주 40시간, 23개 액션 중 13개 동시 진행 시 실행 불가

### §4.8 위험 (Risk)
- Kill Criteria 8개 정의 (K1~K8) — unit_economics.json 박제
- K2(M+1 9.8%), K6(AOV 1,964원 여유) 경계
- K8(현금흐름) 미모니터링 → ACT-11로 가동
