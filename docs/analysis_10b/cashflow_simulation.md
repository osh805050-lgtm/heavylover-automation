### 수식

```
월 총이익 = 월 매출 × 마진율
월 현금흐름 = 월 총이익 - 월 고정비 - 광고비 추가분 - 시리얼 초도비(해당 월만)
누적 현금 = 전월 누적 현금 + 월 현금흐름
```

> 시작 잔고(버퍼): 2,277만원  
> 광고비 추가분 = 당월 총 광고비 - 기준 690만원 (현재 광고비 초과분만 계산)  
> 기준 690만원은 이미 현재 현금흐름에 반영된 상태이므로 추가분만 현금 감소 요인으로 처리  

> **현재 기준 월 현금흐름 계산**  
> 3,800만 × 31% - 656만 = 1,178만 - 656만 = **522만원/월** (추정)  
> ※ 이 값은 광고비 ROI로 매출에 이미 반영되어 있는 구조. 따라서 "추가 광고비"만 별도 차감.

### K8 발동 조건 (공통)

**모든 시나리오에서 K8 발동 트리거는 하나다: 시리얼 OEM 초도비 3,000만원 + 7월 매출 기대치 미달.**

- 초도비 1,500만원 수준이면 어떤 시나리오에서도 K8 미발동.
- 초도비 3,000만원 + 7월 도시락 매출 3,500만 이하(=기대치 -15%) 시 **K8 발동 가능성 高**.
- 특히 시나리오 D(광고 1,200만 스케일) + 초도비 3,000만 + ROAS 디케이(2.8 수준) 동시 발생 시 7월 잔고 계산:
  - ROAS 2.8 시 도시락 매출: 5,483만 → 4,630만 (약 -850만)
  - 7월 현금흐름 = (4,630만+900만)×31% - 656만 - 810만 = 1,711만 - 656만 - 810만 = **245만원**
  - 6월 말 잔고 200만 + 245만 = **445만원** → 이 수준은 1개월 고정비도 못 버팀. K8 경보 지속.

## 4. 결론 및 권고

### 핵심 3줄

1. **시리얼 OEM 초도비 확인이 모든 계산의 선결 조건.** 1,500만원이냐 3,000만원이냐에 따라 K8 발동 가능성이 0%와 50%+로 갈린다. 오늘 나비야에 전화 30분이면 확인 가능.

2. **광고 스케일업과 시리얼 출시를 6월에 동시 집행하면 안전 마진이 1,700만원까지 떨어진다.** 2개월 고정비(1,312만원)밖에 안 남는다. 외부 충격(Meta 계정 정지, 배송 이슈, ROAS 급락) 1회면 K8 진입.

3. **권고 순서**: 시리얼 초도비 실측(1일) → 상생자금 현황 확인(1일) → 초도비 ≤ 1,500만이면 시나리오 C(광고 900만) 선택 / 초도비 > 2,000만이면 광고 스케일 1단계(900만) 6~7월 미루고 시리얼 안착 후 실행.
